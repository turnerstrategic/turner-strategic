# memory/

The organization's working institutional memory — day-to-day record-keeping, as
distinct from the formal governing documents in `institution/`.

- **`archive/`** — superseded material, kept for the record rather than deleted.
- **`briefs/`** — working briefs (problem framings, proposals in progress).
- **`committee/`** — committee notes and decisions.
- **`research/`** — research notes and source material.
- **`lessons/`** — retrospectives and lessons learned.

Empty as of the initial infrastructure scaffold — populated as real work happens.
