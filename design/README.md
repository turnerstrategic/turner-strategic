# design/

Design source material, as distinct from `packages/design-system` (the _engineered_
tokens/theme consumed by code). This directory holds the human-facing source files
those tokens are derived from.

- **`assets/`** — source design assets: logos, icons, imagery.
- **`templates/`** — reusable design templates (decks, documents, etc.).

Empty as of the initial infrastructure scaffold.
